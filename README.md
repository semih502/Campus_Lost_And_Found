🕵️‍♂️ Campus Lost & Found System
A digital platform for university communities to report, track, and recover lost items efficiently.

🚀 Overview
Traditional lost-and-found methods are disorganized. This project provides a secure web application for students and staff to manage lost/found advertisements with integrated messaging and status tracking .  

🛠️ Technical Stack
Core: PHP 8, MySQL, AJAX.  
UI/UX: HTML5, CSS3, JavaScript.  
Security: SHA Encryption, Secure Session Management.  
DB Design: Relational model using Oracle (Barker) Notation.  

💎 Key Features
Authentication: Specific portals for Students and Staff.  
Item Listings: Category-based ads with location data and photo uploads.  
Messaging: Private internal chat between finder and owner.  
Advanced DB: Implements supertype/subtype entities and hierarchical categories .

📊 Database Entities
Includes 9 optimized tables: USERS, ITEMS, CATEGORIES, LOCATIONS, STATUSES, ITEM_PHOTOS, and MESSAGES .
